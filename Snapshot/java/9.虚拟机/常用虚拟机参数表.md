| 虚拟机参数                     | 作用                        |
| ------------------------------ | --------------------------- |
| Xms                            | 初始化堆内存                |
| Xmx                            | 最大堆内存                  |
| XX:+HeapDumpOnOutOfMemoryError | dump出出错内存详细信息      |
| Xss                            | 虚拟机栈容量                |
| PermSize                       | 方法区初始化容量            |
| PermSizeMax                    | 方法区最大容量              |
| -XX:MaxDirectMemorySize        | 直接内存容量，默认与Xms相同 |
|                                |                             |
|                                |                             |
